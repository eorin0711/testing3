HTML의 구조

< !Doctype html >

현재 문서가 HTML5로 작성된 웹문서라는 것을 의미한다

< html lang = "ko" > < /html >

검색사이트에서 특정 언어로 제한해 검색할 때 필요함
예를 들어 한글로 검색 시 lang="ko"로 설정된 웹문서를 우선 검색함

< head > < /head >

웹브라우저가 웹문서를 해석하는 데 필요한 정보를 입력하는 구간

  • < meta > 태그

    • '데이터에 관한 데이터'
    웹문서와 관련된 정보를 지정

    • < meta charset = "UTF-8" >

      metacharset은 화면에 글자를 표시할 때 어떤 인코딩을 사용할지 지정하는 역할을 함
      UTF-8은 유니코드 인코딩 방식의 일종의 문자세트

    • 웹문서의 키워드, 설명, 소유자/제작자 지정

      < meta name = "keywords" content = "html의 구조" > : 키워드
      < meta name = "description" content = "html의 구조 알아보기" > : 설명
      < meta name = "author" content = "Sumin Ham" > : 소유자/제작자

< body > < /body >

실제로 웹브라우저 화면에 나타나는 내용을 적는다
